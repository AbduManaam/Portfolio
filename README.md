# Abdu Manam — Portfolio Website

A modern, animated personal portfolio website built with vanilla HTML, CSS, and JavaScript. Designed to showcase my skills, projects, and experience as a Full Stack Developer specializing in Golang and React.

🌐 **Live Site:** [abdumanaam.github.io](https://abdumanaam.github.io)

---

## ✨ Features

- Custom animated cursor
- Smooth page loader with progress bar
- GSAP-powered scroll animations
- Responsive design for all screen sizes
- Sections: Hero, About, Experience, Skills, Projects, Contact
- Contact form with email integration
- Marquee skill ticker

## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/AbduManaam/AbduManaam.github.io.git
   ```

2. Open the project folder:
   ```bash
   cd AbduManaam.github.io
   ```

3. Open `index.html` in your browser — no build step needed!

> 💡 For best results, use a local server (e.g. VS Code Live Server extension) to avoid CORS issues with local file paths.

---

## 📦 Deployment

This is a static site — no build tools or dependencies required.

### GitHub Pages (Recommended)
1. Push all files to a repo named `AbduManaam.github.io`
2. Go to **Settings → Pages → Branch: main → Save**
3. Live at `https://abdumanaam.github.io`

### Netlify
1. Drag and drop the project folder at [netlify.com](https://netlify.com)
2. Instantly live with a public URL

### Vercel
1. Import the GitHub repo at [vercel.com](https://vercel.com)
2. Click Deploy — done!
